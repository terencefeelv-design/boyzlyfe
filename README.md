# FEELV — bOYz Lyf£

Site officiel de Feel Vlone (FEELV). Album **bOYz Lyf£** — pre-save **01.06.2026**, drop **06.06.2026**.

## Stack

- HTML / CSS / JS vanilla — zéro framework, zéro build
- Three.js (CDN r128) pour la scène 3D du hero
- SVG inline pour la trajectoire A.VIB
- Polices : Bebas Neue + DM Mono (Google Fonts)

